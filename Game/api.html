The Fighting Infantry

This game is about using directional movement to navigate the board before the opponent reaches the player's goal line. Using the the direction keys, and either
getting Powerups or Powerdowns, will determine the speed or life of the player.

Function Parameters Return Value Description
createGrid() row, col N/A Creates the grid dynamicly based on parameters passed.
startGame() N/A N/A Starts the game.
updateScore() N/A count.toString(); Updates the score (Increment/Decrement).
clearRow() N/A N/A Emptys the grids contents.
randomPiece() N/A rng Fills each tile with a random piece (powerup/down).
loadPowerups() N/A N/A Spawns the randomly generated powerups from randompiece onto the grid.
findPowerup() N/A N/A Consumes powerup the player steps on, and places affect on player.
moveLeft() N/A N/A Moves character left.
moveRight() N/A N/A Moves character right.
moveDown() N/A N/A Moves character down.
moveUp() N/A N/A Moves character up.
createBlock() row, col N/A Creates player character
createEnemy() row, col, enemySpeed N/A Spawns an enemy. A new one is generated when current reaches end of grid, or killed.
enemyMove() N/A hPos decides on which random direction enemy will move in next.
setEnemySpeed() enemySpeed N/A Sets a new speed for the enemy.
getEnemySpeed() N/A enemySpeed Returns the value of the enemys current speed.
setHealthBar() health2 N/A Sets current health for player.
getHealthBar() N/A health Gets current health.
shoot() N/A N/A Shoots a bullet across the screen when player presses X button and has the gun powerup, bullet will kill enemy if collides.
getGif() N/A currentGIF Returns current gif to display on screen.
getGameOver() N/A gameOver Returns gameOver, if gameOver = true then show end screen.
getFinTime() N/A finTime Calculate how long the player took to finish the game.
